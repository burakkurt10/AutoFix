## 📌 Overview
**AutoFix** is an interactive and modern iOS application specifically designed and developed for the global Apple "Swift Student Challenge". It aims to revolutionize how drivers interact with car dashboard warnings by leveraging Augmented Reality and Artificial Intelligence.

## 🚀 Key Features
* **AI Smart Scanner (OCR):** Integrates the **Vision Framework** and **AVFoundation** to scan, detect, and analyze dashboard warning lights in real-time using the device camera.
* **3D AR Modeling:** Uses **ARKit** to display interactive 3D vehicle models for a more immersive and interactive learning experience.
* **Modern Onboarding:** Engages users with interactive UI tutorials and walkthroughs powered by **TipKit**.
* **DIY Quick Fixes:** Provides practical "Do-It-Yourself" mechanical solutions and a comprehensive encyclopedia of automotive warning alerts.

## 🛠️ Technical Architecture & Stack
* **Frameworks:** SwiftUI, Vision (Machine Learning/OCR), ARKit (Augmented Reality), AVFoundation (Camera & Audio), TipKit (Onboarding), PhotosUI
* **Language:** Swift
